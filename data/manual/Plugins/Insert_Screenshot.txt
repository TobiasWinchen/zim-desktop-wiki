====== Insert Screenshot ======

This plugin adds a dialog that allows you to take a screenshot of the desktop or of a particular window and directly insert the image into the current zim page.

**Dependencies:** This plugin requires one if the supported screenshot tools to be installed.

Supported tools:

Linux / unix:

* gnome-screenshot
* import (ImageMagick)
* scrot
* spectacle (KDE)

Windows:
* boxcutter

===== Preferences =====

The preference **Screenshot Command** allows you to select the tool to use. **Default choice** selects either Window/region or Fullscreen in the dialog. **Default delay** sets the default delay in the dialog. If the preference **Hide options dialog** is checked, the dialog is not shown and the default values are used.
